Приложение с фоновым сервисом для телефона, которое восстанавливает возможность отладки Bluetooth ADB моста, отсутствующую на стороне телефона для устройств WearOS 3.

В то время как на часах по-прежнему используется опция "Отладка по Bluetooth" в системных настройках, это приложение отвечает за телефонную сторону. С его помощью вы легко можете выбрать сопряженные часы, инициировать Bluetooth рукопожатие RFCOMM и открыть adb-hub локально.

Основные возможности:
• Простой выбор целевых часов из списка сопряженных Bluetooth устройств на вашем телефоне.
• Фоновый сервис постоянно работает, поддерживая логику моста и защиту от закрытия системой.
• Дополнительный TCP порт прослушивания (по умолчанию 44444, настраивается от 1 до 65535).
• Трехстороннее отображение статусов для сервиса, хоста и цели с цветовым различием.
• Информация о конечных точках и команды подключения скрыты для удобства, копируются в буфер обмена в одно нажатие.
• Понятные уведомления об ошибках подключения, занятости портов или отсутствии разрешений.

Два способа подключения:
1. Классический adb-hub: На телефоне создается локальный абстрактный сокет. Подключение осуществимо через переадресацию портов adb forward.
2. Прослушивание TCP: При включении приложение слушает указанный порт. Инструменты на самом телефоне (например, Termux) или другие устройства в локальной сети могут подключаться к часам напрямую по IP-адресу даже без Wi-Fi.

Внимание: версии WearOS старше 3.0 полностью удалили отладку по Bluetooth. Данный инструмент работает только с версиями системы WearOS 2-3. Если в параметрах разработчика ваших часов нет опции "Отладка по Bluetooth", ваша система не поддерживается.
